# wonderful_editor
学習用Qiita風デモアプリ

# ツール・ライブラリの名前

Ruby on Railsの学習のためQiitaを参考にしたデモアプリを作成しました。


## 簡単な説明

ユーザー登録機能
記事の下書き、本番投稿機能
ユーザー関係なく記事の閲覧

ができます。

## 動作環境
Ruby: 3.1.3
Rails: 6.1.7.1
Node 14.20.0
Vue.js: 2.6.11
DB: PostgreSQL
